# Library Management System

Library Management System is a Spring Boot application for managing books, members, and borrowing workflows.

## Installation

Clone the repository and build using Maven:

```bash
git clone https://github.com/your-username/library-management.git
cd library-management
mvn clean install
```

### Usage

Run the application:
```
mvn spring-boot:run
```
### Default URL:
```
http://localhost:8080
```
## Features
- Manage books and catalog records
- Manage library members
- Track borrowing and returns
- Web UI using Thymeleaf
- REST APIs using Spring Web
- Database integration using Spring Data JPA
